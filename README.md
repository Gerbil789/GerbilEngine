# GerbilEngine

Game Engine hobby project

## Getting Started

#### Setup
1. Clone the Project
2. Navigate to the Scripts Folder
3. Generate project files by executing script
    - Windows: `Win-GenerateProjects.bat`
    - Linux: `Not supported`
    - Mac: `Not supported`
4. Build **`Editor`** project
